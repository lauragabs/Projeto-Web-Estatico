# Laura's Kitchen

Projeto de website estatico desenvolvido para a disciplina, utilizando apenas HTML e CSS.

## Sobre o projeto

O site tem como tema culinaria e compartilha:
- receitas
- dicas de cozinha
- informacoes sobre o blog

## Requisitos atendidos

- 4 paginas web:
  - index.html
  - receitas.html
  - dicas.html
  - sobre.html
- Estrutura com Header, Main e Footer em todas as paginas
- Menu de navegacao repetido em todas as paginas
- Efeito hover no menu com CSS
- Layout responsivo para desktop e smartphone

## Estrutura de pastas

- index.html
- receitas.html
- dicas.html
- sobre.html
- assets/
  - css/
    - style.css
  - img/

## Tecnologias utilizadas

- HTML5
- CSS3
